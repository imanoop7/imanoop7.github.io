Sample Book Title

This is an example book structure to demonstrate how you can organize your e-books with chapters and PDF downloads.

How to Use This Structure

  1. Create a folder for each book under content/e-books/
  2. Add an _index.md file for the book overview
  3. Create individual chapter files with frontmatter
  4. Include PDF download links in chapters
  5. Use the chapter parameter to order chapters
